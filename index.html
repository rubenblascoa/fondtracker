Preguntas Frecuentes

¿Cómo añade FondTracker los datos de mis fondos? +

Utilizamos el ISIN para buscar en nuestro catálogo de más de 490 fondos españoles y ETFs. Los precios se actualizan mediante Yahoo Finance API o nuestro scraper nativo de QueFondos para fondos no cotizados.

¿Es segura mi información financiera? +

Absolutamente. No almacenamos tus credenciales bancarias. Solo registramos los fondos y participaciones que tú nos indiques. Tus datos están protegidos por encriptación AES-256 y un sistema de autenticación JWT.

¿Cómo funciona el servicio de WhatsApp? +

Integramos el servicio CallMeBot para enviarte un resumen diario a la hora que elijas (por defecto 9:00 y 18:00). Solo necesitas configurar tu número y API key en tu panel de ajustes.
